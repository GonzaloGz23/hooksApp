# 05-hooks-app

Este es un proyecto simple de React, creado con Vite y TypeScript, que utiliza TailwindCSS para los estilos. Su propósito principal es servir como una herramienta de **aprendizaje y exploración de los diferentes hooks de React**. Estoy utilizándolo para comprender mejor cómo funcionan y cómo aplicarlos en diversas situaciones.

## Características

- Aplicación React moderna.
- Desarrollado con TypeScript para una mayor robustez.
- Vite para un desarrollo rápido y eficiente.
- Estilado con TailwindCSS para un diseño responsivo y personalizable.
- Ejemplos prácticos y sencillos del uso de hooks de React para fines educativos.

## Requisitos

Antes de empezar, asegúrate de tener Node.js y npm (o Yarn) instalados en tu sistema.

- Node.js (v18 o superior)
- npm (v9 o superior) o Yarn (v1.22 o superior)

## Instalación

Sigue estos pasos para configurar el proyecto localmente:

1.  Clona el repositorio:

    ```bash
    git clone https://github.com/GonzaloGz23/hooksApp.git
    cd hooksApp
    ```

2.  Instala las dependencias:
    ```bash
    npm install
    # o
    yarn install
    ```

## Uso

### Ejecutar en modo desarrollo

Para iniciar el servidor de desarrollo:

```bash
npm run dev
# o
yarn dev
```
